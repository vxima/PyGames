# PyGames

Repositório com jogos feitos no Pygame 

## Para rodar é necessario criar o ambiente virtual antes :

### Linux
```bash
$ virtualenv env3
```

```bash
$ source env3/bin/activate
```

```bash
$ pip install -r requirements.txt
```

### Windows
```bash
$ python -m venv .venv
```
```bash
$ pip install -r requirements.txt --user
```
